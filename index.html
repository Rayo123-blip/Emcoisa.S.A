Extraccion de Oro

Lixiviación de Oro

Un proceso químico para extraer oro de minerales a través de una solución de cianuro.

Flotación de Oro

Proceso que utiliza la diferencia de densidad para separar el oro de otros minerales.

Amalgamación de Oro

Proceso en el que el oro se mezcla con mercurio para formar una amalgama.
